## API

### [textarea[vts-input]]

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| disabled | Disable textarea | `boolean` | `false` |
| vtsAutosize | Auto adjust height of textarea | `boolean` or `(minRow: number, maxRow: number) ` | `false` |
